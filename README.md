# חופשת החלומות של משפחת חפץ · אטלנטיס דובאי

מדריך אינטראקטיבי פרטי (נפתח ישירות בדפדפן, בלי התחברות).

## קובצי מפה ל-Google My Maps
מייבאים כל קובץ כשכבה נפרדת:
1. נכנסים ל-<https://www.google.com/mymaps> → "צור מפה חדשה".
2. בכל שכבה: "ייבוא" → בוחרים קובץ CSV.
3. עמודות למיקום: "כתובת" · לשם הסיכה: "שם".
4. מומלץ: שכבה `00-base-always-on` דלוקה תמיד, ולכל יום שכבה נפרדת שאפשר לכבות/להדליק.

הקבצים: base, day1–day5.
